# 自我介绍模板

## 版本一：30 秒自我介绍

适用问题：

请你简单介绍一下自己

先做个简短自我介绍

你先介绍一下你的背景

模板：

我是一名有 [X] 年经验的 [岗位]，主要方向是 [领域]。最近几年的工作重点是 [职责或场景]。我做过比较有代表性的项目有 [项目 A] 和 [项目 B]，主要产出是 [结果或价值]。这次我想找的岗位，核心是希望在 [目标方向] 上继续发挥经验，同时进一步提升 [成长方向]。

## 版本二：1 分钟自我介绍

模板：

我一开始是从 [起点岗位或方向] 进入这个行业，过去 [X] 年里逐步聚焦在 [当前方向]。我平时做得比较多的是 [系统 / 产品 / 业务领域]。在我最近一份工作里，我主要负责 [范围]，包括 [关键职责]。对我来说最有代表性的项目是 [项目]，当时最大的挑战是 [挑战]，最后拿到的结果是 [结果或指标]。我现在想找的岗位，希望既能继续用上我在 [已有经验] 上的积累，也能让我进一步成长到 [下一阶段能力]。

## 版本三：2 到 3 分钟自我介绍

结构建议：

1. 先讲当前岗位和职责范围

2. 再讲职业发展的主线

3. 挑两个最有代表性的项目展开

4. 总结自己的核心优势

5. 说明为什么现在出来看机会

6. 说明为什么这个岗位适合自己

## 定制化提示

如果是创业公司岗位，优先强调：

- 交付速度
- 业务理解
- 独立推动
- 在不确定环境下做事情

如果是大公司岗位，优先强调：

- 系统稳定性
- 规模与复杂度
- 跨团队协作
- 规范和工程质量

如果是 AI 相关岗位，优先强调：

- 业务场景理解
- 数据质量和评估
- Prompt 或工作流设计
- 模型接入与落地经验

## 注意事项

不要把团队成果直接说成全部是你做的。

不要说自己全链路负责，除非你真的从需求到上线都主导过。
